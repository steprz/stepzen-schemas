## Setting up a Linked DEV and GitHub API

Please add the appropriate configurations to your configuration folder.

```bash
configurationset:
  - configuration:
      name: dev_config
      username: 'YOUR_DEV_USERNAME'
  - configuration:
      name: github_config
      Authorization: 'Bearer YOUR_GITHUB_ACCESS_TOKEN'

```
